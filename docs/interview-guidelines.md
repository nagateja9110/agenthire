# AgentHire Interview Guidelines

## Structure

Interviews focus on the skills the candidate actually matched against the job's
requirements. Each round combines conceptual questions, one practical coding task, and
a rubric-based evaluation. Questions must be grounded in the job's required and
preferred skills - never in technologies absent from both the job and the resume.

## Rubric

Evaluators grade each criterion on a four-level scale: strong_no, no, yes, strong_yes.
Every required skill gets its own rubric criterion, plus one criterion for communication
and problem decomposition. A hire recommendation requires "yes" or better on all
required-skill criteria.

## Coding tasks

Coding tasks should be completable in 45 minutes, use the candidate's matched stack,
and assess real-world patterns (state management, API integration, error handling)
rather than puzzle tricks.
